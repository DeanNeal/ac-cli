## Installation
[![NPM](https://nodei.co/npm/ace-cli.png)](https://nodei.co/npm/ac-cli/)

Using npm:

```bash
npm i -g ac-cli
```

Generate component:

```bash
ac-cli g <path> <component name>

example

ac-cli g app example

there will be 2 files created
./app/example.component.js
./app/example.component.html

```
